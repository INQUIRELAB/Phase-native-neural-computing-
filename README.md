## 📖 Abstract
This repository presents a MATLAB implementation of the Unified Complex Neuron (UCN) framework for modeling phase-dependent neural dynamics in biological systems. The provided simulations demonstrate the capability of UCN to capture phase locking, stochastic perturbation responses, and regime-dependent neural activity using experimentally recorded mouse hippocampal datasets. The framework enables reproducible evaluation of phase-integrator neuron models and their advantages over conventional spiking neuron formulations.

## 🧪 Simulations

To reproduce the results, run the following scripts individually:

- **Simulation 1:** `PLT_Manual.m`
- **Simulation 2:** `PLT_COMPARE.m`
- **Simulation 3:** `PLT_RND_PHASE.m`
- **Simulation 4:** `PLT_RND_PHASE2.m`
- **Simulation 5:** `REGIME_SELECTOR_Rem.m`
- **Simulation 6:** `REGIME_SELECTOR_Wake.m`

Ensure that all required datasets are placed in the working directory before execution.
